# AutomationExercise – Authentication Test Cases

This repository contains manual test cases for the **Authentication module (Register, Login, Logout)** of the AutomationExercise web application.

## Scope
The test cases are designed based on the actual UI and behavior of the AutomationExercise website and cover:
- User Registration
- User Login
- User Logout

## Test Case Design
The test cases use a **test case–level result approach**, where:
- Test steps describe user actions
- Expected Result, Actual Result, and Status are recorded at the **test case level**
- Test data is defined for each test case
- Detailed steps are maintained without duplicating result entries per step

## Test Scenario Structure
- **TS.Auth.001** – Register
- **TS.Auth.002** – Login
- **TS.Auth.003** – Logout

Each test case follows this ID format:
TS.Auth.<Scenario>.<TestCase>
